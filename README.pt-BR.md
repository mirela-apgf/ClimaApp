🌤️ Aplicativo de Clima do Brasil

Aplicativo mobile desenvolvido utilizando Java e XML para exibir informações climáticas de cidades do Brasil.
O app oferece uma interface simples e intuitiva, permitindo que os usuários consultem rapidamente as condições atuais do tempo.

<br>
🚀 Funcionalidades

- Busca de clima por cidades brasileiras
- Informações climáticas em tempo real
- Interface limpa e intuitiva

<br>
🛠 Tecnologias

- Java
- XML (Layouts Android)
- Android Studio

API de Clima

<br>
💡 Usabilidade

A aplicação foi projetada com foco em simplicidade e facilidade de uso.
Os usuários podem buscar uma cidade e visualizar instantaneamente suas condições climáticas, tornando o app prático para o dia a dia.
